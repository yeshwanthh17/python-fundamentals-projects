# Text Analyzer (Python)

## üìå Description
Text Analyzer is a simple Python program that analyzes a block of text and provides
useful textual insights such as character count, word statistics, sentence count,
and frequency analysis.

This project is built using **core Python fundamentals only** and focuses on
strings, lists, dictionaries, loops, conditions, and functions.

---

## ‚ú® Features
The program performs the following analysis on a given text:

1. Counts total number of characters
2. Counts total number of words
3. Counts total number of sentences
4. Displays frequency of each character (case-sensitive)
5. Displays frequency of each word (case-insensitive)
6. Finds the most frequent word
7. Finds the least frequent word
8. Finds the first non-repeating character

---

## üßæ Input
- A single block of text
- The text may contain:
  - uppercase and lowercase letters
  - spaces
  - punctuation marks such as `. , ! ?`

The input can be provided either:
- directly inside the program, or
- through standard input (`input()`)

---

## üì§ Output
The program prints clearly labeled results for all analyses, including:

- Total Characters
- Total Words
- Total Sentences
- Character Frequency
- Word Frequency
- Most Frequent Word
- Least Frequent Word
- First Non-Repeating Character
